# 美术资源说明

## 📁 目录结构

```
assets/
├── backgrounds/    # 背景图片
├── characters/     # 角色立绘
├── audio/          # 音效/BGM
└── fonts/          # 字体（可选）
```

---

## 🖼️ 背景图片

### 规格要求

| 属性 | 要求 |
|------|------|
| **格式** | JPG 或 PNG |
| **尺寸** | 1920x1080（推荐）或任意 16:9 比例 |
| **大小** | < 2MB/张 |
| **风格** | 写实或手绘均可 |

### 第一关：出租屋

- **文件名**：`backgrounds/rental-room.jpg`
- **描述**：一间普通的出租屋，要有：
  - 钢琴（靠墙）
  - 厨房区域（可见刀具）
  - 地板（可有血迹）
  - 窗户（开着）
  - 花瓶碎片（地上）

### 占位图

开发期间可以使用纯色或简单图形占位：
- 使用在线工具生成占位图
- 或用 CSS 渐变代替

---

## 👤 角色立绘

### 规格要求

| 属性 | 要求 |
|------|------|
| **格式** | PNG（透明背景） |
| **尺寸** | 200x200（头像）或 400x600（全身） |
| **大小** | < 500KB/张 |

### 第一关：房东太太

- **文件名**：`characters/woman.png`
- **描述**：30 多岁女性，神色紧张

### 占位图

开发期间可以：
- 使用 emoji 代替 👩
- 使用简单几何图形
- 暂时留空（代码已处理）

## 🔊 音效/BGM

### 建议音效

| 场景 | 音效 |
|------|------|
| 点击疑点 | 轻微"叮"声 |
| 选项选择 | 按钮点击声 |
| 怀疑值增加 | 紧张音效 |
| 胜利结局 | 轻松音乐 |
| 失败结局 | 恐怖音效 |

### 建议 BGM

| 场景 | 音乐 |
|------|------|
| 主界面 | 悬疑氛围音乐 |
| 游戏中 | 紧张背景音乐 |
| 结局 | 根据胜负播放不同音乐 |
